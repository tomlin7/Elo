## Combat Pass Star Economy

Elo players level up the Combat Pass using earned seasonal currency: **Combat Stars (CS)**:

- **Earning Stars**: Players earn CS stars dynamically upon match completions (3 stars for wins, 1 star for losses/participating) and completing daily challenges.
- **Tiers Progression**: The pass spans a 50-tier progression track requiring **10 stars** to tier up.
- **Tiers Ledger**: Claims are logged transactionally using a bitmask tracker to prevent double-claiming rewards.
- **Theme Unlocks**: Active rewards alternate between free credits and premium theme files (Neon Ghost, Cyberpunk, Spectral Aura) from the cosmetics vault.
